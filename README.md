# What If

A real-time counterfactual viewer for live football. Watch any live football match. At any moment in the broadcast, ask **"what if?"** — the system reads the actual frames of the broadcast, finds the moment you mean, and uses Veo to generate the alternative reality as a video clip that continues from the real footage. The live match keeps playing the whole time. Walk away with a shareable director's cut of your what-ifs at full time.

Built on **GMI Cloud** (real-time vision-language perception), **RocketRide** (multi-agent runtime), and **Google's Gemini + Veo** (deep video understanding and counterfactual video generation).
